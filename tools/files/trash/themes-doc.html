info Overview

The Material Icons OS uses a server-based theme system that loads themes dynamically from configuration files. Each theme consists of:

  • CSS file - Scoped overrides using data-theme attributes
  • JavaScript file (optional) - Canvas animations with mandatory cleanup
  • Assets - Images, fonts, and other resources

check_circle Key Features

  • Dynamic Loading - Themes load without page reload
  • Canvas Support - Animated backgrounds with proper cleanup
  • CSS Variables - Easy color theming with custom properties
  • Scoped Selectors - Prevent style bleeding between themes
  • Hot Reload - Apply themes via URL or Settings UI
  • Resource Cleanup - Automatic cleanup of animations, timers, listeners

warning 🚨 Critical Rules (Read First!)

error These rules are non-negotiable

  1. CSS MUST be scoped with body[data-theme="your-theme"] to prevent style bleeding
  2. Scripts MUST set window.ThemeCleanup = function cleanup() {...} at global scope
  3. Canvas MUST stay transparent - use clearRect(), never fillRect() with color
  4. Event listeners MUST be removed in cleanup using stored references
  5. Animation frames MUST be cancelled with cancelAnimationFrame()

lightbulb Pro Tip: Test cleanup early

Apply your theme, then immediately switch to another theme. If animations continue or styles persist, your cleanup is broken. Fix it before adding features.

brush CSS: Scoping & Variables

error CSS Scoping is Mandatory

Always wrap theme-specific rules in body[data-theme="your-theme"]. Unscoped rules will leak to other themes.

/* ✅ CORRECT: Scoped with data attribute */
body[data-theme="your-theme"] {
  --bg-color: linear-gradient(135deg, #1a1a2e, #0f3460);
}

body[data-theme="your-theme"] .icon {
  background: var(--glass-bg);
  border: 1px solid var(--glass-border);
}

/* ❌ WRONG: Global selector will leak */
.icon {
  background: red; /* Affects ALL themes! */
}

code JavaScript & Canvas Animations

error Script Requirements (Non-Negotiable)

  1. Wrap entire script in IIFE: (function() { ... })();
  2. Set window.ThemeCleanup = function cleanup() {...} at global scope
  3. Keep canvas transparent: use clearRect(), never fill with color
  4. Store event handler references for proper removal in cleanup
  5. Cancel all requestAnimationFrame, setInterval, setTimeout

Complete Template with Best Practices

/**
 * Your Theme Name - Animation Script
 * ✅ Follows all cleanup requirements
 */
(function() {
  'use strict';
  
  // 1. Get canvas with defensive checks
  const canvas = document.getElementById('themeCanvas');
  if (!canvas) {
    console.warn('⚠ Canvas not found - theme running in no-canvas mode');
    window.ThemeCleanup = function cleanup() {};
    return;
  }

  const ctx = canvas.getContext('2d');
  if (!ctx) {
    console.error('❌ Could not get canvas context');
    window.ThemeCleanup = function() {};
    return;
  }

  // 2. Track ALL resources for cleanup
  const themeResources = {
    animationFrame: null,
    resizeHandlers: [],
    intervals: [],
    timeouts: [],
    canvas: canvas,
    ctx: ctx
  };

  // 3. Helper: Register resize handler with tracking
  function addResizeHandler(handler) {
    window.addEventListener('resize', handler);
    themeResources.resizeHandlers.push(handler);
  }

  // 4. HiDPI-aware resize handling
  function resizeCanvas() {
    const dpr = window.devicePixelRatio || 1;
    canvas.width = window.innerWidth * dpr;
    canvas.height = window.innerHeight * dpr;
    canvas.style.width = window.innerWidth + 'px';
    canvas.style.height = window.innerHeight + 'px';
    ctx.setTransform(1, 0, 0, 1, 0, 0); // Reset
    ctx.scale(dpr, dpr); // Scale for sharp rendering
  }
  resizeCanvas();
  addResizeHandler(resizeCanvas);

  // 5. Ensure canvas stays behind UI
  canvas.style.cssText = 'position:fixed;top:0;left:0;width:100%;height:100%;z-index:0;pointer-events:none;display:block;background:transparent;';

  // 6. Optional: Respect reduced motion preference
  const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches;
  if (prefersReducedMotion) {
    console.log('♿ Reduced motion preferred - skipping animation');
    window.ThemeCleanup = function() {};
    return;
  }

  // 7. Your animation code
  function animate() {
    // CRITICAL: Clear with transparency
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    
    // Draw your animation here
    // Example: ctx.fillStyle = 'rgba(0,255,65,0.1)'; ctx.fillRect(...);
    
    // Track animation frame with ThemeManager
    themeResources.animationFrame = requestAnimationFrame(animate);
    if (typeof ThemeManager !== 'undefined' && ThemeManager.trackAnimation) {
      ThemeManager.trackAnimation(themeResources.animationFrame);
    }
  }
  
  // Start animation
  console.log('✓ Your theme animation started');
  animate();

  // 8. ✅ REQUIRED: Set global cleanup function
  window.ThemeCleanup = function cleanup() {
    console.log('🔴 Your theme cleanup started');
    
    // Cancel animation frame
    if (themeResources.animationFrame) {
      cancelAnimationFrame(themeResources.animationFrame);
      themeResources.animationFrame = null;
    }
    
    // Remove all resize listeners
    themeResources.resizeHandlers.forEach(handler => {
      try { window.removeEventListener('resize', handler); } catch(e) {}
    });
    themeResources.resizeHandlers = [];
    
    // Clear intervals/timeouts
    themeResources.intervals.forEach(id => { try { clearInterval(id); } catch(e) {}; });
    themeResources.intervals = [];
    themeResources.timeouts.forEach(id => { try { clearTimeout(id); } catch(e) {}; });
    themeResources.timeouts = [];
    
    // Clear canvas thoroughly
    try {
      ctx.setTransform(1, 0, 0, 1, 0, 0);
      ctx.clearRect(0, 0, canvas.width, canvas.height);
    } catch (e) { console.warn('⚠ Canvas clear error:', e); }
    
    // Free memory
    // largeArrays.length = 0; etc.
    
    console.log('✓ Your theme cleanup complete');
  };
  
  // 9. Optional: Register via cleanup registry
  if (typeof ThemeManager !== 'undefined' && ThemeManager.registerCleanup) {
    ThemeManager.registerCleanup(() => {
      console.log('✓ Your theme registry cleanup');
    });
  }
  
  console.log('🟢 Your theme script loaded successfully');
})();

cleaning_services Cleanup System

The ThemeManager provides a robust cleanup system to prevent memory leaks and visual artifacts when switching themes.

How Cleanup Works

  1. Theme loads → Script sets window.ThemeCleanup function
  2. User switches theme → ThemeManager calls cleanup function
  3. Cleanup executes → Cancels animations, removes listeners, clears canvas
  4. New theme loads → Clean slate, no artifacts from previous theme

ThemeManager Cleanup Helpers

Method Purpose Example
ThemeManager.trackAnimation(id) Track requestAnimationFrame for auto-cancel ThemeManager.trackAnimation(rafId)
ThemeManager.trackInterval(id) Track setInterval for auto-clear ThemeManager.trackInterval(intervalId)
ThemeManager.trackTimeout(id) Track setTimeout for auto-clear ThemeManager.trackTimeout(timeoutId)
ThemeManager.registerCleanup(fn) Register additional cleanup function ThemeManager.registerCleanup(() => {...})

error Cleanup Checklist (Must Complete All)

  • ✅ Set window.ThemeCleanup function (not just return it)
  • ✅ Cancel all requestAnimationFrame calls
  • ✅ Clear all setInterval and setTimeout timers
  • ✅ Remove all addEventListener listeners using stored references
  • ✅ Clear canvas with clearRect() and resetTransform()
  • ✅ Remove any DOM elements created by the theme
  • ✅ Free large arrays/objects to help garbage collection
  • ✅ Log cleanup start/end for debugging

verified Best Practices

1. Use CSS Variables for Maintainability

/* ✅ GOOD: Uses variables */
body[data-theme="your-theme"] .window {
  background: var(--window-bg);
  color: var(--text-primary);
}

/* ❌ BAD: Hardcoded colors */
.window {
  background: #ffffff; /* Won't adapt to theme */
}

2. Maintain Canvas Transparency

// ✅ GOOD: Clear canvas (preserves background)
ctx.clearRect(0, 0, canvas.width, canvas.height);

// ❌ BAD: Fill with color (blocks CSS background)
ctx.fillStyle = 'rgba(0, 0, 0, 0.5)';
ctx.fillRect(0, 0, canvas.width, canvas.height);

3. Handle HiDPI Displays

function resizeCanvas() {
  const dpr = window.devicePixelRatio || 1;
  canvas.width = window.innerWidth * dpr;
  canvas.height = window.innerHeight * dpr;
  canvas.style.width = window.innerWidth + 'px';
  canvas.style.height = window.innerHeight + 'px';
  ctx.setTransform(1, 0, 0, 1, 0, 0);
  ctx.scale(dpr, dpr);
}

4. Debounce Resize Handlers

let resizeTimeout;
function onResize() {
  clearTimeout(resizeTimeout);
  resizeTimeout = setTimeout(() => {
    resizeCanvas();
    // Re-init animation if needed
  }, 150);
}
window.addEventListener('resize', onResize);
// In cleanup: clearTimeout(resizeTimeout);

5. Optimize Animation Performance

// ✅ Limit particle count
const MAX_PARTICLES = 100;

// ✅ Use object pooling for reusable particles
const particlePool = [];
function getParticle() {
  return particlePool.pop() || new Particle();
}

// ✅ Pause animation when tab hidden
document.addEventListener('visibilitychange', () => {
  if (document.hidden && animationFrame) {
    cancelAnimationFrame(animationFrame);
  } else {
    animate();
  }
});

warning ⚠️ Common Pitfalls

❌ Style Bleeding (Most Common)

Problem

Theme CSS affects other themes after switching

Solution

/* Always scope with data attribute */
body[data-theme="your-theme"] .icon { ... }

❌ Canvas Blocking Background

Problem

Can't see CSS background through canvas

Solution

  • Use clearRect() not fillRect()
  • Set canvas CSS: background: transparent !important
  • Ensure canvas z-index is 0 (behind UI)

❌ Memory Leaks from Uncleaned Resources

Problem

Animations continue after theme switch

Solution

  • ✅ Set window.ThemeCleanup at global scope
  • ✅ Store handler references for removeEventListener
  • ✅ Cancel all animation frames/intervals/timeouts
  • ✅ Use ThemeManager.track*() helpers

❌ Low CSS Specificity

Problem

Theme styles overridden by base CSS

Solution

/* Use high-specificity selectors */
html body[data-theme="your-theme"],
body[data-theme="your-theme"] {
  background: ... !important;
}

❌ Not Testing Cleanup

Problem

Theme works initially but breaks on switch

Solution

Test cleanup immediately: Apply theme → Switch to another → Check console for errors or visual artifacts.

bug_report Troubleshooting

Background Not Showing

Problem

Background image/color not visible

Solutions

  1. Check CSS specificity - use body[data-theme="..."] + !important
  2. Verify image path is correct (check Network tab for 404)
  3. Ensure canvas is transparent: background: transparent !important
  4. Check if base styles override - inspect computed styles

Animations Not Stopping (Memory Leak)

Problem

Previous theme animation continues after switching

Solutions

  1. ✅ Verify window.ThemeCleanup is set at global scope
  2. ✅ Cancel animation frames with cancelAnimationFrame()
  3. ✅ Remove event listeners with stored references
  4. ✅ Use ThemeManager.trackAnimation() for auto-cancel
  5. ✅ Clear canvas with clearRect() and resetTransform()

Theme Not Loading

Problem

Theme doesn't appear in Settings

Solutions

  1. Validate themes.json syntax (use JSONLint)
  2. Verify file paths are relative to desktop.html
  3. Check browser console for CORS or 404 errors
  4. Clear browser cache or add ?reload=1 to URL

CSS Not Applying

Problem

Theme styles not showing

Solutions

  1. Increase specificity with body[data-theme="..."]
  2. Add !important to critical overrides temporarily for testing
  3. Check if CSS file is loading (Network tab → Status 200)
  4. Verify data-theme attribute is set on <body>

themes/your-theme/script.js (with cleanup)

(function() {
  'use strict';
  
  const canvas = document.getElementById('themeCanvas');
  if (!canvas) { window.ThemeCleanup = function() {}; return; }
  
  const ctx = canvas.getContext('2d');
  const resources = { animationFrame: null, resizeHandlers: [] };
  
  function resizeCanvas() {
    const dpr = window.devicePixelRatio || 1;
    canvas.width = window.innerWidth * dpr;
    canvas.height = window.innerHeight * dpr;
    canvas.style.width = window.innerWidth + 'px';
    canvas.style.height = window.innerHeight + 'px';
    ctx.setTransform(1, 0, 0, 1, 0, 0);
    ctx.scale(dpr, dpr);
  }
  resizeCanvas();
  window.addEventListener('resize', resizeCanvas);
  resources.resizeHandlers.push(resizeCanvas);

  function animate() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    // Your animation here
    resources.animationFrame = requestAnimationFrame(animate);
    if (ThemeManager?.trackAnimation) ThemeManager.trackAnimation(resources.animationFrame);
  }
  animate();

  // ✅ REQUIRED: Global cleanup function
  window.ThemeCleanup = function cleanup() {
    if (resources.animationFrame) cancelAnimationFrame(resources.animationFrame);
    resources.resizeHandlers.forEach(h => window.removeEventListener('resize', h));
    ctx.setTransform(1, 0, 0, 1, 0, 0);
    ctx.clearRect(0, 0, canvas.width, canvas.height);
  };
  
  if (ThemeManager?.registerCleanup) ThemeManager.registerCleanup(() => console.log('✓ Cleanup'));
})();
