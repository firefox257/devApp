info Overview

The Material Icons OS uses a server-based theme system that loads themes dynamically from configuration files. Each theme consists of:

  • CSS file - Overrides default styles and defines visual appearance
  • JavaScript file (optional) - Handles canvas animations and dynamic effects
  • Assets - Images, fonts, and other resources

check_circle Key Features

  • Dynamic Loading - Themes load from server without localStorage
  • Canvas Support - Optional animated backgrounds with proper cleanup
  • CSS Variables - Easy color theming with CSS custom properties
  • Data Attributes - Theme switching via data-theme attributes
  • Hot Reload - Apply themes via URL parameters or Settings UI
  • Resource Cleanup - Automatic cleanup of animations, timers, and event listeners

brush CSS Overrides

warning CSS Specificity Strategy

Themes use data attributes for specificity. Always use this pattern:

/* ✅ CORRECT: High specificity with data attribute */
html body[data-theme="your-theme"],
body[data-theme="your-theme"] {
  background: url('themes/your-theme/background.png') !important;
  background-size: cover !important;
}

/* ❌ WRONG: Low specificity (will be overridden) */
body {
  background: red;
}

code JavaScript & Canvas Animations

lightbulb Script Requirements

Theme scripts MUST follow this pattern for proper cleanup:

Basic Template with Cleanup

/**
 * Your Theme Name - Animation Script
 * ✅ Compatible with ThemeManager cleanup system
 */
(function() {
  'use strict';
  
  // 1. Get canvas with error handling
  const canvas = document.getElementById('themeCanvas');
  if (!canvas) {
    console.warn('Your theme: #themeCanvas not found');
    window.ThemeCleanup = function cleanup() {};
    return;
  }

  const ctx = canvas.getContext('2d');
  
  // 2. Track resources for cleanup
  const themeResources = {
    animationFrame: null,
    resizeHandlers: [],
    intervals: [],
    timeouts: []
  };

  // 3. Helper: Register event listeners for cleanup
  function addResizeHandler(handler) {
    window.addEventListener('resize', handler);
    themeResources.resizeHandlers.push(handler);
  }

  // 4. Handle resizing
  function resizeCanvas() {
    canvas.width = window.innerWidth;
    canvas.height = window.innerHeight;
  }
  resizeCanvas();
  addResizeHandler(resizeCanvas);

  // 5. Your animation code here
  function animate() {
    // Clear canvas (IMPORTANT: Keep transparent!)
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    
    // Draw your animation
    // ...
    
    // Track animation frame with ThemeManager
    themeResources.animationFrame = requestAnimationFrame(animate);
    if (typeof ThemeManager !== 'undefined' && ThemeManager.trackAnimation) {
      ThemeManager.trackAnimation(themeResources.animationFrame);
    }
  }
  
  // Start animation
  console.log('✓ Your theme animation started');
  animate();

  // 6. EXPOSE CLEANUP VIA window.ThemeCleanup (REQUIRED)
  window.ThemeCleanup = function cleanup() {
    console.log('✓ Your theme cleanup started');
    
    // Cancel animation frame
    if (themeResources.animationFrame) {
      cancelAnimationFrame(themeResources.animationFrame);
      themeResources.animationFrame = null;
    }
    
    // Remove all resize listeners
    themeResources.resizeHandlers.forEach(handler => {
      try { window.removeEventListener('resize', handler); } catch(e) {}
    });
    themeResources.resizeHandlers = [];
    
    // Clear intervals/timeouts
    themeResources.intervals.forEach(id => { try { clearInterval(id); } catch(e) {}; });
    themeResources.intervals = [];
    themeResources.timeouts.forEach(id => { try { clearTimeout(id); } catch(e) {}; });
    themeResources.timeouts = [];
    
    // Clear canvas thoroughly
    try {
      ctx.clearRect(0, 0, canvas.width, canvas.height);
      ctx.resetTransform();
    } catch (e) { console.warn('⚠ Error clearing canvas:', e); }
    
    console.log('✓ Your theme cleanup complete');
  };
  
  // 7. Optional: Register via cleanup registry for multiple cleanup functions
  if (typeof ThemeManager !== 'undefined' && ThemeManager.registerCleanup) {
    ThemeManager.registerCleanup(() => {
      console.log('✓ Your theme registry cleanup');
    });
  }
  
})();

error Critical Rules

  • ✅ DO: Use IIFE (Immediately Invoked Function Expression)
  • ✅ DO: Set window.ThemeCleanup = function cleanup() {...} at top level
  • ✅ DO: Clear canvas with clearRect() (don't fill with color)
  • ✅ DO: Remove event listeners in cleanup
  • ✅ DO: Use ThemeManager.trackAnimation/Interval/Timeout() for auto-cleanup
  • ❌ DON'T: Fill canvas with solid color (blocks background)
  • ❌ DON'T: Return cleanup instead of setting window.ThemeCleanup
  • ❌ DON'T: Leave animation frames running after cleanup

cleaning_services Cleanup System

The ThemeManager provides a robust cleanup system to prevent memory leaks and visual artifacts when switching themes.

How Cleanup Works

  1. Theme loads → Script sets window.ThemeCleanup function
  2. User switches theme → ThemeManager calls cleanup function
  3. Cleanup executes → Cancels animations, removes listeners, clears canvas
  4. New theme loads → Clean slate, no artifacts from previous theme

ThemeManager Cleanup Helpers

Method Purpose Example
ThemeManager.trackAnimation(id) Track requestAnimationFrame for auto-cancel ThemeManager.trackAnimation(rafId)
ThemeManager.trackInterval(id) Track setInterval for auto-clear ThemeManager.trackInterval(intervalId)
ThemeManager.trackTimeout(id) Track setTimeout for auto-clear ThemeManager.trackTimeout(timeoutId)
ThemeManager.registerCleanup(fn) Register additional cleanup function ThemeManager.registerCleanup(() => {...})

Resource Tracking Pattern

// Track all resources in one object
const themeResources = {
  animationFrame: null,
  resizeHandlers: [],
  intervals: [],
  timeouts: []
};

// When creating animation frame
themeResources.animationFrame = requestAnimationFrame(animate);
if (ThemeManager?.trackAnimation) {
  ThemeManager.trackAnimation(themeResources.animationFrame);
}

// In cleanup function
window.ThemeCleanup = function cleanup() {
  if (themeResources.animationFrame) {
    cancelAnimationFrame(themeResources.animationFrame);
  }
  // ... clear other resources
};

Complete Cleanup Checklist

task_alt Every theme script should:

  • ✅ Set window.ThemeCleanup function (not just return it)
  • ✅ Cancel all requestAnimationFrame calls
  • ✅ Clear all setInterval and setTimeout timers
  • ✅ Remove all addEventListener listeners
  • ✅ Clear canvas with clearRect() and resetTransform()
  • ✅ Remove any DOM elements created by the theme
  • ✅ Free large arrays/objects to help garbage collection
  • ✅ Log cleanup start/end for debugging

api Theme System API

ThemeManager Object

The global ThemeManager object provides theme control:

// Get current theme
ThemeManager.currentTheme  // → "rainy-forest"

// Get theme configuration
ThemeManager.themeConfig   // → { "default": {...}, "matrix": {...} }

// Apply a theme
await ThemeManager.apply('matrix');

// Get available themes
ThemeManager.getAvailableThemes();
// → [{id: 'default', name: 'Default Blue'}, ...]

// Populate a select element
await ThemeManager.populateThemeSelector('themeSelector');

// Cleanup helpers for theme scripts
ThemeManager.trackAnimation(rafId);
ThemeManager.trackInterval(intervalId);
ThemeManager.trackTimeout(timeoutId);
ThemeManager.registerCleanup(cleanupFn);

verified Best Practices

1. Use CSS Variables

/* ✅ GOOD: Uses variables */
.window {
  background: var(--window-bg);
  color: var(--text-primary);
  border: 1px solid var(--border-color);
}

/* ❌ BAD: Hardcoded colors */
.window {
  background: #ffffff;
  color: #333;
  border: 1px solid #ccc;
}

2. Maintain Transparency

// ✅ GOOD: Clear canvas
ctx.clearRect(0, 0, canvas.width, canvas.height);

// ❌ BAD: Fill with color (blocks background)
ctx.fillStyle = 'rgba(0, 0, 0, 0.5)';
ctx.fillRect(0, 0, canvas.width, canvas.height);

3. Always Set window.ThemeCleanup

// ✅ GOOD: Set global cleanup function
window.ThemeCleanup = function cleanup() {
  cancelAnimationFrame(animationFrame);
  window.removeEventListener('resize', handler);
};

// ❌ BAD: Just return (won't be captured by script src)
return function cleanup() {
  // This return value is ignored when loading via <script src>
}

4. Use High Specificity

/* ✅ GOOD */
html body[data-theme="your-theme"] {
  background: url(...) !important;
}

/* ❌ BAD: Overridden by base styles */
body {
  background: url(...);
}

5. Optimize Performance

// ✅ GOOD: Limit particle count
const particleCount = 100;

// ❌ BAD: Too many particles
const particleCount = 1000;

// ✅ GOOD: Use requestAnimationFrame for smooth animation
function animate() {
  // draw code
  animationFrame = requestAnimationFrame(animate);
  if (ThemeManager?.trackAnimation) {
    ThemeManager.trackAnimation(animationFrame);
  }
}

6. Handle Errors Gracefully

if (!canvas) {
  console.warn('Canvas not found');
  window.ThemeCleanup = function() {};
  return;
}

// In cleanup, wrap DOM operations in try/catch
try {
  ctx.clearRect(0, 0, canvas.width, canvas.height);
} catch (e) {
  console.warn('Cleanup error:', e);
}

7. Test Across Screen Sizes

window.addEventListener('resize', () => {
  canvas.width = window.innerWidth;
  canvas.height = window.innerHeight;
});

// And remove in cleanup!
window.ThemeCleanup = function cleanup() {
  window.removeEventListener('resize', resizeHandler);
};

bug_report Troubleshooting

Background Not Showing

Problem

Background image/color not visible

Solutions

  1. Check CSS specificity - use !important and data attributes
  2. Verify image path is correct
  3. Check browser console for 404 errors
  4. Ensure canvas is transparent: background: transparent !important

Canvas Blocking Background

Problem

Can't see background through canvas

Solutions

  1. Use clearRect() not fillRect()
  2. Don't fill with semi-transparent color
  3. Set canvas CSS: background: transparent !important

Animations Not Stopping (Memory Leak)

Problem

Previous theme animation continues after switching

Solutions

  1. ✅ Set window.ThemeCleanup (not just return)
  2. ✅ Cancel animation frames with cancelAnimationFrame()
  3. ✅ Remove event listeners with removeEventListener()
  4. ✅ Use ThemeManager.trackAnimation() for auto-cancel
  5. ✅ Clear canvas with clearRect() and resetTransform()

Theme Not Loading

Problem

Theme doesn't appear in Settings

Solutions

  1. Check themes.json syntax (valid JSON)
  2. Verify file paths are correct
  3. Check browser console for errors
  4. Ensure CORS headers if loading from different domain

CSS Not Applying

Problem

Theme styles not showing

Solutions

  1. Increase specificity with data attributes
  2. Use !important for critical overrides
  3. Check if CSS file is loading (Network tab)
  4. Clear browser cache

themes/your-theme/script.js (with cleanup)

(function() {
  'use strict';
  
  const canvas = document.getElementById('themeCanvas');
  if (!canvas) {
    window.ThemeCleanup = function() {};
    return;
  }
  
  const ctx = canvas.getContext('2d');
  const themeResources = { animationFrame: null, resizeHandlers: [] };
  
  function resizeCanvas() {
    canvas.width = window.innerWidth;
    canvas.height = window.innerHeight;
  }
  resizeCanvas();
  window.addEventListener('resize', resizeCanvas);
  themeResources.resizeHandlers.push(resizeCanvas);

  function animate() {
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    // Your animation here
    themeResources.animationFrame = requestAnimationFrame(animate);
    if (ThemeManager?.trackAnimation) {
      ThemeManager.trackAnimation(themeResources.animationFrame);
    }
  }
  
  animate();

  // ✅ REQUIRED: Set global cleanup function
  window.ThemeCleanup = function cleanup() {
    if (themeResources.animationFrame) cancelAnimationFrame(themeResources.animationFrame);
    themeResources.resizeHandlers.forEach(h => window.removeEventListener('resize', h));
    ctx.clearRect(0, 0, canvas.width, canvas.height);
    ctx.resetTransform();
  };
  
  // Optional: Register additional cleanup
  if (ThemeManager?.registerCleanup) {
    ThemeManager.registerCleanup(() => console.log('✓ Cleanup complete'));
  }
})();
