Tile Drawer Module Test

This page tests the `tileDrawer.js` module. You'll need to have `tileDrawer.js` in the same directory (or adjust the import path). A simple spritesheet named 'spritesheet.png' is assumed to be in the same directory. If you don't have one, you can create a 64x64 image with 4 distinct 32x32 tiles. For example: - Top-left: Red square - Top-right: Green square - Bottom-left: Blue square - Bottom-right: Yellow square
