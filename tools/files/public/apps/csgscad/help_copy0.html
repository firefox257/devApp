1. About

Welcome to the CSG Editor! This is a powerful, mobile-friendly, browser-based CAD tool that lets you create complex 3D models using Constructive Solid Geometry (CSG) operations.

You write scripts in a JavaScript-like syntax to define shapes (like cubes, spheres, cylinders), transform them (translate, rotate, scale), and combine them using boolean operations (union, difference, intersection). The result is rendered in real-time in the 3D view.

This editor is designed for rapid prototyping, 3D printing preparation, and educational purposes. It combines the power of OpenSCAD-style scripting with the immediacy of a live 3D preview.

Coordinate Systems: 2D paths use Y-up (X=right, Y=up). 3D geometry uses Z-up (X=width, Y=depth, Z=height).

5. Paths: How to Create Them

2D paths are the foundation for complex extruded shapes. Paths are defined as arrays of commands with parameters.

Path Data Format

{
  path: [
    'm', x, y,           // Move to (Absolute)
    'lr', dx, dy,        // Line to (Relative)
    'q', cx, cy, x, y,   // Quadratic Bezier
    'x', cx, cy, x, y,   // Arc via control point
    'n', segments,       // Set segments for next curve
  ],
  fn: 40 // Default resolution for curves
}

Path Commands Reference

CmdTypeSyntaxDescription
mAbsolute'm', x, yMove To (absolute position)
mrRelative'mr', dx, dyMove To (relative offset)
lAbsolute'l', x, yLine To (absolute)
lrRelative'lr', dx, dyLine To (relative)
qAbsolute'q', cx, cy, x, yQuadratic Bezier curve
qrRelative'qr', dcx, dcy, dx, dyQuadratic Bezier (relative)
cAbsolute'c', cx1, cy1, cx2, cy2, x, yCubic Bezier curve
crRelative'cr', dcx1, dcy1, dcx2, dcy2, dx, dyCubic Bezier (relative)
xAbsolute'x', cx, cy, x, yArc via control point
xrRelative'xr', dcx, dcy, dx, dyArc (relative)
nUtility'n', segmentsSet segments for next command
r3D Modifier'r', angleSet target rotation for extrusion
s3D Modifier's', sx, sySet target scale for extrusion

Automatic Hole Detection

The shape() function automatically detects holes based on path nesting using bounding box containment and scanline testing. Outer contours are solids; nested contours are holes.

arcPath3d() Helper

Generates smooth 3D arc paths for extrusion. Parameters: startAng, endAng (degrees), r or d (radius/diameter), fn (resolution).

Path Examples

// Simple Triangle
const trianglePath = {
  path: ['m', 0, 0, 'l', 10, 0, 'l', 5, 10, 'l', 0, 0],
  fn: 32
};
const triangleShape = shape(trianglePath);
const straightPath = { path: ['m', 0, 0, 0, 'l', 0, 0, 2], fn: 1 };
return linePaths3d(triangleShape, straightPath);

// Using Path2d class (method chaining)
const roundedRect = new Path2d()
  .fn(32)
  .m(0, 0)
  .l(50, 0)
  .q(55, 0, 55, 5)  // Rounded corner
  .l(55, 25)
  .q(55, 30, 50, 30)
  .l(0, 30)
  .q(-5, 30, -5, 25)
  .l(-5, 5)
  .q(-5, 0, 0, 0)
  .getPoints();

⚡ CSG Operations

union() CSG

union(...target)
Description: Performs boolean union operation on multiple meshes using three-bvh-csg. Combines all meshes into a single mesh, removing internal geometry where meshes overlap. Uses ADDITION operation.
ParameterTypeDescription
target...THREE.Mesh|BrushTwo or more meshes to combine (spread operator). Can also pass arrays with spread: union(...array)
Returns: THREE.Mesh|Brush - Combined mesh result from CSG evaluator, or null if no meshes provided
// Example Usage:
const box1 = cube([5, 5, 5]);
const box2 = translate([3, 0, 0], cube([5, 5, 5]));
const combined = union(box1, box2);  // Two overlapping boxes merged

// Can also pass arrays
const allParts = [part1, part2, part3];
const assembly = union(...allParts);

// Complex assembly
const base = cube([20, 20, 2]);
const post1 = translate([5, 5, 2], cylinder({ r: 2, h: 10 }));
const post2 = translate([15, 15, 2], cylinder({ r: 2, h: 10 }));
const assembly = union(base, post1, post2);

difference() CSG

difference(meshes, ...target)
Description: Performs boolean subtraction operation using three-bvh-csg. Subtracts one or more meshes from a base mesh, creating holes or cutouts. Uses SUBTRACTION operation.
ParameterTypeDescription
meshesTHREE.Mesh|Brush|ArrayBase mesh(es) to subtract from. Can be single mesh or array of meshes (auto-unioned)
target...THREE.Mesh|BrushMesh(es) to subtract (spread operator). These are the "cutters"

// Multiple holes

  block,
  translate([3, 3, -2.5], hole),
  translate([-3, -3, -2.5], clone(hole))
);

// Array of cutters
const cutters = [hole1, hole2, hole3];
const perforated = difference(basePlate, ...cutters);

subdivide() Transform

subdivide({ resolution = 0.2 }, ...target)
Description: Subdivides mesh geometry to increase resolution. Iteratively splits faces until all edge lengths are below specified resolution. Uses edge-length-based subdivision with vertex deduplication.
ParameterTypeDefaultDescription

target...THREE.Mesh|Brush-Mesh(es) to subdivide

// Lattice unit function
function latticeUnit(size) {
  const strut = cylinder({ r: 1, h: size, fn: 8 });
  const xStrut = rotate([0, 90, 0], strut);
  const yStrut = rotate([90, 0, 0], strut);
  const zStrut = strut;
  return union([
    xStrut, 
    translate([0, size/2, 0], yStrut), 
    translate([0, 0, size/2], zStrut)
  ]);
}

const unitSize = 10, count = 3;
for (let x = 0; x < count; x++) {
  for (let y = 0; y < count; y++) {
    for (let z = 0; z < count; z++) {
      const unit = latticeUnit(unitSize);
      units.push(translate([x * unitSize, y * unitSize, z * unitSize], unit));
    }
  }
}

const innerFrame = cube([count * unitSize - 2, count * unitSize - 2, count * unitSize - 2]);
