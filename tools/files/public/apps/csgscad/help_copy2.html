1. About

Welcome to the CSG Editor! This is a powerful, mobile-friendly, browser-based CAD (Computer-Aided Design) tool that lets you create complex 3D models using Constructive Solid Geometry (CSG) operations.

You write scripts in a JavaScript-like syntax to define shapes (like cubes, spheres, cylinders), transform them (translate, rotate, scale), and combine them using boolean operations (union, difference, intersection). The result is rendered in real-time in the 3D view.

This editor is designed for rapid prototyping, 3D printing preparation, and educational purposes. It combines the power of OpenSCAD-style scripting with the immediacy of a live 3D preview.

4.3 CSG Scripts vs Editor Scripts

  • CSG Scripts (Red Monitor): Must return 3D objects for rendering
  • Editor Scripts (Blue Editor): Return reusable functions/constants

How to Use get() and include()

  • include(name): Load exports from an Editor Script tab
  • get(name): Load 3D model result from a CSG Script tab

External get() and include()

  • include(name, filepath): Load Editor script from file
  • get(name, filepath): Load CSG script from file

How Paths Work

  1. Absolute: Start with /
  2. Library: Start with $lib/ (uses Library Path setting)
  3. Relative: Relative to current project file directory

5. Paths: How to Create Them

2D paths are the foundation for complex extruded shapes.

Path Format

{
path: [
'm', x, y,           // Move to (Absolute)
'lr', dx, dy,        // Line to (Relative)
'q', cx, cy, x, y,   // Quadratic Bezier
'x', cx, cy, x, y,   // Arc via control point
],
fn: 40 // Resolution for curves
}

2D Coordinates: Y-up system (positive X=right, positive Y=up)

Path Commands Reference

CommandTypeSyntaxDescription
mAbsolute'm', x, yMove To
mrRelative'mr', dx, dyMove To (relative)
lAbsolute'l', x, yLine To
lrRelative'lr', dx, dyLine To (relative)
qAbsolute'q', cx, cy, x, yQuadratic Bezier
qrRelative'qr', dcx, dcy, dx, dyQuadratic Bezier (relative)
cAbsolute'c', cx1, cy1, cx2, cy2, x, yCubic Bezier
crRelative'cr', dcx1, dcy1, dcx2, dcy2, dx, dyCubic Bezier (relative)
xAbsolute'x', cx, cy, x, yArc via control point
xrRelative'xr', dcx, dcy, dx, dyArc (relative)
nUtility'n', segmentsSet segments for next command
r3D Modifier'r', angleRotation for extrusion
s3D Modifier's', sx, syScale for extrusion

Automatic Hole Detection

The shape() function automatically detects holes based on path nesting.

arcPath3d() Helper

Generates smooth 3D paths for extrusion. Parameters: startAng, endAng (degrees), r (radius), fn (resolution).

Path Examples

Simple Triangle
const trianglePath = {
path: ['m', 0, 0, 'l', 10, 0, 'l', 5, 10, 'l', 0, 0]
};
const triangleShape = shape(trianglePath);
const straightPath = { path: ['m', 0, 0, 0, 'l', 0, 0, 2], fn: 1 };
return linePaths3d(triangleShape, straightPath);

6. scadCSG Commands and How to Use Them

3D Coordinate System: Z-up system (X=width, Y=depth, Z=height)

Primitive Shapes

FunctionParametersDescription
sphere(){ r, d, fn }Creates a sphere (r=radius, d=diameter, fn=segments)
cube()[x, y, z]Creates a box with specified dimensions
cylinder(){ r, rt, rb, d, dt, db, h, fn }Creates cylinder/cone (rt/rb=top/bottom radius)
Primitive Examples
// Sphere with radius 5mm, 32 segments
const ball = sphere({ r: 5, fn: 32 });

// Cube 20x20x10mm
const box = cube([20, 20, 10]);

// Cone (different top/bottom radii)
const cone = cylinder({ rt: 0, rb: 5, h: 15, fn: 32 });

return union([ball, box, cone]);

Transformations

FunctionParametersDescription
translate()[x, y, z], ...targetMoves geometry by offset
rotate()[x, y, z], ...targetRotates in degrees (X, Y, Z order)
scale()[x, y, z], ...targetScales by factors
scaleTo(){ x, y, z }, ...targetScales proportionally to target dimension
scaleAdd(){ x, y, z }, ...targetAdds value to dimensions
floor()...targetMoves lowest point to Z=0
align(){ lx, rx, cx, uy, dy, cy, bz, tz, cz }, ...targetAligns by bounding box
placement()offsets, obj, ...targetAligns targets relative to obj

Placement Commands: [REF-ANCHOR][TARGET-ALIGN][AXIS]

  • X: l=left(min), r=right(max), c=center (llx, lrx, lcx, rlx, rrx, rcx, clx, crx, ccx)
  • Y: d=down(min), u=up(max), c=center (ldy, luy, lcy, rdy, ruy, rcy, cdy, cuy, ccy)
  • Z: b=bottom(min), t=top(max), c=center (lbz, ltz, lcz, rbz, rtz, rcz, cbz, ctz, ccz)
Transformation Examples
// Translate 10mm up on Z
const moved = translate([0, 0, 10], cube([5, 5, 5]));

// Rotate 45° around Z
const rotated = rotate([0, 0, 45], cube([5, 5, 5]));

// Scale to exactly 20mm on X (proportional)
const sized = scaleTo({ x: 20 }, cube([10, 10, 10]));

// Add 5mm to all dimensions
const expanded = scaleAdd({ x: 5, y: 5, z: 5 }, cube([10, 10, 10]));

// Place sphere on top of cube with 2mm offset
const base = cube([20, 20, 5]);
const top = sphere({ r: 5 });
placement({ 'ttz': 2 }, base, top);

return union([base, top]);

CSG Operations

FunctionParametersDescription
union()...targetCombines objects (addition)
difference()mainMesh, ...targetSubtracts targets from main
intersect()...targetFinds overlapping volume (2+ meshes)
inverseIntersect()...targetSymmetric difference (XOR)
CSG Examples
// Union: Combine objects
const combined = union(cube([10, 10, 10]), sphere({ r: 5 }));

// Difference: Cut hole through cube
const base = cube([20, 20, 20]);
const cutter = cylinder({ r: 3, h: 25 });
const withHole = difference(base, translate([10, 10, -2.5], cutter));

// Intersect: Keep only overlap
const overlap = intersect(cube([15, 15, 15]), sphere({ r: 10 }));

// XOR: Keep non-overlapping parts
const xor = inverseIntersect(cube([15, 15, 15]), sphere({ r: 10 }));

return withHole;

Path Classes (NEW)

Path2d and Path3d are class-based path builders with method chaining.

ClassMethodsDescription
Path2d m(), mr(), l(), lr(), q(), qr(), c(), cr(), x(), xr(), n(), fn(), path(), getPoints(), boundingBox() 2D path builder
Path3d All Path2d + m(x,y,z), r(rotation), s(sx,sy), close(v), xyInitAng(v) 3D path builder
Path Class Examples
// Path2d: Rounded rectangle
const path2d = new Path2d()
.fn(32)
.m(0, 0)
.l(50, 0)
.l(50, 30)
.l(0, 30)
.l(0, 0);

// Path3d: Extrusion path with rotation
const path3d = new Path3d()
.fn(16)
.m(0, 0, 0)
.r(0)
.l(0, 0, 10)
.r(90)
.l(10, 0, 10)
.close(true);

// Get processed points
const points2d = path2d.getPoints();
const points3d = path3d.getPoints();

// Convert between 2D and 3D
const path2dFrom3d = new Path2d().path(path3d);
const path3dFrom2d = new Path3d().path(path2d);

return path2d;

2D to 3D Extrusion

FunctionParametersDescription
shape()shapeDataPathGenerates THREE.Shape array with hole detection
linePaths3d()target, commandPathExtrudes Shape along 3D path (scale/rotation support)
extrude3d()target, commandPathExtrudes Path2d along 3D path (advanced)
sweep3d()circularPath, wallPath, holeWallPathSweeps geometry along circular path
arcPath3d(){ startAng, endAng, r, d, fn }Helper for curved 3D paths
Extrusion Examples
// Simple straight extrusion
const profile = new Path2d()
.fn(32)
.m(0, 0)
.l(10, 0)
.l(10, 10)
.l(0, 10)
.l(0, 0);

const extrusionPath = new Path3d()
.fn(1)
.m(0, 0, 0)
.l(0, 0, 5);

const shapes = shape(profile);
const extruded = linePaths3d(shapes, extrusionPath);

// Curved extrusion
const curvedPath = arcPath3d({
startAng: 0,
endAng: 90,
r: 20,
fn: 32
});
const curvedExtrusion = linePaths3d(shapes, curvedPath);

// Sweep for tubes/rings
const circularPath = new Path2d().fn(64).m(20, 0).l(20, 0);
const wallProfile = new Path2d().fn(16).m(0, 0).l(0, 3).l(2, 3).l(2, 0).l(0, 0);
const swept = sweep3d(circularPath, wallProfile, null);

return extruded;

Path Manipulation Examples
const base = new Path2d().fn(32).m(0, 0).l(10, 0).l(10, 10).l(0, 10).l(0, 0);

// Translate 5mm right, 3mm up
const moved = translatePath([5, 3], new Path2d().path(base));

// Rotate 45 degrees
const rotated = rotatePath(45, new Path2d().path(base));

// Scale to 50mm width
const sized = scaleToPath({ x: 50 }, new Path2d().path(base));

// Add 5mm to all sides
const expanded = scaleAddPath({ x: 5, y: 5 }, new Path2d().path(base));

// Get bounding box
const bbox = boundingBoxPath(base);

// Convert between 2D and 3D
const path3d = convertTo3d(base, 0);
const path2d = convertTo2d(path3d);

return base;

3D Imports (Requires await)

All import functions are asynchronous! Must use await.

FunctionParametersReturns
importStl()filepathTHREE.Mesh
importObj()filepathBrush[]
importGlb()filepathBrush[]
importFbx()filepathBrush[]
importSvg()filepathPath2d[]
Import Examples
// Import STL and combine
async function model() {
const imported = await importStl("$lib/part.stl");
const base = cube([30, 30, 5]);
return union([imported, floor(base)]);
}

// Import SVG and extrude
async function logo() {
const svgPaths = await importSvg("$lib/logo.svg");
const shapes = svgPaths.map(p => shape(p));
const extrusionPath = new Path3d().m(0, 0, 0).l(0, 0, 2);
const extruded = shapes.flatMap(s => linePaths3d(s, extrusionPath));
return union(extruded);
}

// Import GLB with multiple meshes
async function assembly() {
const parts = await importGlb("$lib/assembly.glb");
return union(parts);
}

Utility Examples

const redCube = color(0xff0000, cube([10, 10, 10]));
const blueSphere = color("#0000ff", sphere({ r: 5 }));

// Convex hull
const hull = convexHull(cube([10, 10, 10]), translate([15, 0, 0], sphere({ r: 5 })));

// Bounding box
const bbox = boundingBox(cube([10, 10, 10]), sphere({ r: 5 }));

// Subdivide for smoother booleans

// Show/hide for debugging
const debug = hide(cube([10, 10, 10]));
const visible = show(sphere({ r: 5 }));

// Deep clone
const original = cube([10, 10, 10]);
const copy1 = clone(original);
const copy2 = clone(original);
translate([15, 0, 0], copy1);
translate([-15, 0, 0], copy2);
const cloned = union([original, copy1, copy2]);

// Expand nested structures
const meshes = expand(union(cube([10, 10, 10]), sphere({ r: 5 })));

return cloned;

7. Complete Examples

Basic Assembly with Placement
// Base plate with mounted components
const basePlate = cube([50, 50, 3]);
const mount1 = cylinder({ r: 5, h: 10 });
const mount2 = cylinder({ r: 5, h: 10 });

// Place mounts on corners
placement({ 'lbl': 0, 'lbd': 0, 'bbz': 0 }, basePlate, mount1);
placement({ 'rbr': 0, 'rbu': 0, 'bbz': 0 }, basePlate, mount2);

// Add holes through mounts
const holeCutter = cylinder({ r: 2, h: 15 });
const mountWithHole1 = difference(mount1, translate([0, 0, -2.5], holeCutter));
const mountWithHole2 = difference(mount2, translate([0, 0, -2.5], clone(holeCutter)));

return union([basePlate, mountWithHole1, mountWithHole2]);
Parametric Box with Wall Thickness
// Parameters
const outerW = 40, outerD = 30, outerH = 20, wallThickness = 2;

// Outer box
const outer = cube([outerW, outerD, outerH]);

// Inner box (subtracted to create hollow)
const inner = cube([
outerW - wallThickness * 2,
outerD - wallThickness * 2,
outerH - wallThickness
]);

// Position inner box
const positionedInner = translate([wallThickness, wallThickness, wallThickness], inner);

// Create hollow box
const hollowBox = difference(outer, positionedInner);

// Add lid
const lid = cube([outerW + 1, outerD + 1, 2]);
const positionedLid = translate([0, 0, outerH], lid);

return union([hollowBox, positionedLid]);
Custom Profile Extrusion
// I-beam profile
const profile = new Path2d()

Modular Design with Clone
// Create base module
function createModule() {
const base = cube([20, 20, 5]);
const post = cylinder({ r: 3, h: 10 });
const positionedPost = translate([10, 10, 5], post);
return union([base, positionedPost]);
}

// Create multiple instances
const module1 = createModule();
const module2 = clone(module1);
const module3 = clone(module1);

// Arrange in row
translate([0, 0, 0], module1);
translate([25, 0, 0], module2);
translate([50, 0, 0], module3);

// Add connecting plate
const plate = cube([75, 20, 2]);
const positionedPlate = translate([0, 0, 15], plate);

return union([module1, module2, module3, positionedPlate]);
Lattice Structure with Subdivide
// Lattice unit
function latticeUnit(size) {
const strut = cylinder({ r: 1, h: size, fn: 8 });
const xStrut = rotate([0, 90, 0], strut);
const yStrut = rotate([90, 0, 0], strut);
const zStrut = strut;
return union([xStrut, translate([0, size/2, 0], yStrut), translate([0, 0, size/2], zStrut)]);
}

// Create lattice array
const units = [];
const unitSize = 10, count = 3;
for (let x = 0; x < count; x++) {
for (let y = 0; y < count; y++) {
for (let z = 0; z < count; z++) {
const unit = latticeUnit(unitSize);
units.push(translate([x * unitSize, y * unitSize, z * unitSize], unit));
}
}
}

// Subdivide for smoother struts
const lattice = subdivide({ resolution: 0.5 }, union(units));

// Add outer frame
const frame = cube([count * unitSize, count * unitSize, count * unitSize]);
const innerFrame = cube([count * unitSize - 2, count * unitSize - 2, count * unitSize - 2]);
const hollowFrame = difference(frame, translate([1, 1, 1], innerFrame));

return union([lattice, hollowFrame]);
SVG Import with Text
async function svgTextModel() {
// Load font
const myFont = await font("$lib/arial.ttf");

// Create text path
const textPath = text({ font: myFont, text: "CSG", fontSize: 10, fn: 32 });

// Load SVG logo
const svgPaths = await importSvg("$lib/logo.svg");

// Combine text and SVG
const textShape = shape(textPath);
const svgShapes = svgPaths.map(p => shape(p));

// Extrude all
const extrusionPath = new Path3d().m(0, 0, 0).l(0, 0, 2);
const textExtruded = linePaths3d(textShape, extrusionPath);
const svgExtruded = svgShapes.flatMap(s => linePaths3d(s, extrusionPath));

// Position and combine
translate([0, 0, 0], textExtruded);
translate([0, 15, 0], svgExtruded);

return union([...textExtruded, ...svgExtruded]);
}
