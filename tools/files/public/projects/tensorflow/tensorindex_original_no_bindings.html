🏗️ Model Architecture

Embedding: Embedding(vocab_size, hidden_dim)
SwiGLU Block ×2:
  ├─ Gate: Dense(hidden) → Swish
  ├─ Value: Dense(hidden)
  └─ Output: Swish(gate) ⊗ value
Output Head: Dense(vocab_size) + softmax

Loss: Categorical Cross-Entropy (next-token)
