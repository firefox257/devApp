🏗️ Model Architecture

Input: Token IDs via chat-aware tokenizer
Special Tokens: <|user|>, <|assistant|>, <|system|>, <|stop|>
Embedding: Embedding(vocab+1, hidden_dim) (+1 for <unk>)
RoPE: Enabled
SwiGLU Blocks ×2: Residual gated MLP
Output Head: Dense(vocab+1) + softmax
Generation: Autoregressive loop with temperature sampling
Optimizer: Adam | Max Seq: 300
Training Mode: Segment-Aware (`<|stop|>` boundaries)
