🎨 Material Icons Documentation

Font Location: /fonts/MaterialIcons-Regular.ttf
Version: Material Icons Regular
Usage: Semantic color-coded icon system for consistent UI/UX

📥 Font Installation

@font-face { font-family: 'Material Icons'; font-style: normal; font-weight: 400; src: url('/fonts/MaterialIcons-Regular.ttf') format('truetype'); font-display: block; }

🎯 Icon Usage

HTML Usage

<span class="material-icon" style="color: #007bff">add_circle</span>

JavaScript/DOM Creation

const icon = document.createElement('span'); icon.className = 'material-icon'; icon.textContent = 'add_circle'; icon.style.color = '#007bff';

🔧 Best Practices

✅ DO:

  • Use semantic colors consistently across the application
  • Always set aria-hidden="true" on decorative icons
  • Use the exact ligature names (case-sensitive)
  • Apply colors via inline styles or CSS classes for consistency
  • Test icons in both light and dark contexts

⚠️ AVOID:

  • Using emoji icons instead of Material Icons
  • Changing semantic colors without updating the design system
  • Forgetting to set font-family on icon elements
  • Using arbitrary color values instead of the defined palette
